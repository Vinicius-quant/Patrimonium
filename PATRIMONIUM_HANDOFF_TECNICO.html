05 Todos os Calculos e Racionais

Cada calculo abaixo esta implementado inline no JSX (useMemo). O backend deve reimplementar como services dedicados.

5.1 Consolidacao Patrimonial (Media Ponderada)

Racional: Cada empresa/conta tem patrimonio e retorno proprios. A consolidacao pondera pelo patrimonio.

// Patrimonio total
const totalPat = co.empresas.reduce((s, e) => s + e.patrimonio, 0);

// Ganho total do mes
const totalGanho = co.empresas.reduce((s, e) => s + e.ganhoMes, 0);

// Retorno mensal ponderado
const wRent = co.empresas.reduce((s, e) => s + e.rentMes * e.patrimonio, 0) / totalPat;

// %CDI ponderado
const wCDI = co.empresas.reduce((s, e) => s + e.pctCDI * e.patrimonio, 0) / totalPat;

// Retorno anual ponderado
const wRentAno = co.empresas.reduce((s, e) => s + e.rentAno * e.patrimonio, 0) / totalPat;
const wCDIAno = co.empresas.reduce((s, e) => s + e.pctCDIAno * e.patrimonio, 0) / totalPat;
Racional
A media ponderada garante que contas maiores tenham mais peso no resultado consolidado. Se uma conta de R$ 8M rende 1.2% e outra de R$ 500k rende 0.5%, o consolidado reflete mais a conta maior.

5.4 IR por Faixa (Imposto de Renda Regressivo)

const irBracket = (dias) =>
  dias <= 180 ? 0.225     // 22.5% ate 6 meses
  : dias <= 360 ? 0.20    // 20.0% ate 1 ano
  : dias <= 720 ? 0.175   // 17.5% ate 2 anos
  : 0.15;                   // 15.0% acima de 2 anos

Racional: Tabela regressiva de IR conforme Lei 11.033/2004. Aplicada a renda fixa, CDB, LF, debentures nao incentivadas.

5.5 Classificacao Tributaria de Fundos

const taxCat = (f) => {
  if (f.isentoIR || ["CRA", "CRI", "LCI", "LCA"].includes(f.tipo)) return "isento";
  if (f.tipo === "Fundos") return "comecotas";

Racional: CRA/CRI/LCI/LCA sao isentos de IR por lei. Fundos sofrem come-cotas semestral. Acoes tem tributacao diferenciada (15% sobre ganho de capital).

// Binary search: encontra remessa mensal para atingir US$ 1M em 60 meses
function solveMonthlyUSD(usdRate) {
  let lo = 0, hi = 50000;  // range: US$ 0 a US$ 50k/mes
  while (hi - lo > 1) {
    const mid = (lo + hi) / 2;
    const final = simulate(mid, usdRate);  // roda 60 meses
    if (final >= targetUSD) hi = mid;
    else lo = mid;
  }
  return hi;
}

13.7 Tool Use — Ferramentas para a IA

A Claude API suporta tool use (function calling): a IA pode chamar funcoes do backend para buscar dados ou executar calculos. Isso e mais preciso que colocar tudo no system prompt.

import Anthropic from '@anthropic-ai/sdk';
import { betaZodTool } from '@anthropic-ai/sdk/helpers/beta/zod';
import { z } from 'zod';

// Definir ferramentas que a IA pode chamar
const calcularIR = betaZodTool({
  name: 'calcular_imposto_resgate',
  description: 'Calcula o IR devido ao resgatar uma posicao especifica',
  inputSchema: z.object({
    positionId: z.string().describe('ID da posicao a resgatar'),
    valorResgate: z.number().optional().describe('Valor parcial (default: total)'),
  }),
  run: async ({ positionId, valorResgate }) => {
    // Buscar posicao no banco via Prisma
    const pos = await prisma.position.findUnique({ where: { id: positionId } });
    const valor = valorResgate || pos.currentValue;
    const ganho = valor * (pos.returnMonth / 100) * (pos.daysToMaturity / 30);
    const aliquota = irBracket(pos.daysToMaturity);
    const imposto = ganho * aliquota;
    return JSON.stringify({ posicao: pos.shortName, valorResgate: valor,
      ganhoEstimado: ganho, aliquota, imposto, liquido: valor - imposto });
  },
});

const simularAplicacao = betaZodTool({
  name: 'simular_aplicacao',
  description: 'Simula o retorno de uma aplicacao com taxa, prazo e valor',
  inputSchema: z.object({
    valor: z.number().describe('Valor a aplicar em R$'),
    taxaAnual: z.number().describe('Taxa anual em % (ex: 14.5)'),
    meses: z.number().describe('Prazo em meses'),
    indexador: z.enum(['CDI', 'IPCA', 'PRE']).optional(),
  }),
  run: async ({ valor, taxaAnual, meses, indexador }) => {
    const taxaMensal = Math.pow(1 + taxaAnual / 100, 1 / 12) - 1;
    const bruto = valor * Math.pow(1 + taxaMensal, meses);
    const ganho = bruto - valor;
    const dias = meses * 30;
    const ir = ganho * irBracket(dias);
    return JSON.stringify({ valorInicial: valor, bruto, ganho, ir,
      liquido: bruto - ir, rentLiquida: ((bruto - ir) / valor - 1) * 100 });
  },
});

const buscarPosicoes = betaZodTool({
  name: 'buscar_posicoes',
  description: 'Busca posicoes da carteira por filtros (classe, tipo, gestora)',
  inputSchema: z.object({
    classe: z.string().optional(),
    tipo: z.string().optional(),
    gestora: z.string().optional(),
    orderBy: z.enum(['valor', 'rentabilidade', 'pctCDI']).optional(),
  }),
  run: async (filters) => {
    const positions = await prisma.position.findMany({
      where: { ...filters, account: { client: { id: clientId } } },
      orderBy: { [filters.orderBy || 'currentValue']: 'desc' },
    });
    return JSON.stringify(positions);
  },
});
